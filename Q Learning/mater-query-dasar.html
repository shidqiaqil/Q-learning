SELECT

Select berfungsi untuk memspesifikasikan nama-nama kolom yang akan ditampilkan. Nama-nama kolom dituliskan setelah klausa ini. Cara penggunaan SELECT adalah sebagai berikut :

SELECT col1, col2, col3, … , coln FROM table_name;

Jika ingin menggunakan SELECT dengan suatu kondisi bisa menggunakan query dengan format seperti dibawah ini:

SELECT col1, col2, col3, … , coln FROM table_name WHERE [condition];

Query diatas berfungsi untuk melihat data yang ada di col1, col2, col3, hingga coln dari tabel yang bernama table_name. Jika ingin melihat semua data yang ada di semua kolom dari table_name bisa menggunakan query seperti dibawah ini:

SELECT * FROM table_name;
